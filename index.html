VLAN Configuration in Cisco Packet Tracer
VLAN allows administrators to limit the access of the user by dividing isolated LAN segments. In other words, Vlan is the process of dividing a group of devices to give limited access to a Network.

VLAN 0, 4095 — These are reserved VLANs that cannot be seen or used.
VLAN 1 — It is the default VLAN of switches. This VLAN can’t be deleted or edited but can be used.
VLAN 2–1001 — This is a normal VLAN range. We can create, edit and delete these VLANs.
VLAN 1002–1005 — These are CISCO defaults for FDDI and token rings. These VLANs can’t be deleted.
